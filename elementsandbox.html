How to Add More Rules & Elements:

1. Add new element type: Define color & add it to the elements list.
2. Define behavior: Update physics & special actions in update().
3. Add interactions: Define how it reacts with others in checkInteractions().
4. Draw properly: Set color and add any special drawing in draw().
5. Update UI: Add element to selection list for placement.

Example: Add "Lightning" element:
- Color: #FFFFAA
- Behavior: moves randomly, limited life
- Interaction: zaps fire to remove it

Use space to cycle elements, R to spawn random elements.
Click or drag on canvas to place elements.
